# forkify
## !!! To run the app you need to have node and npm installed in your computer
You can install it here https://nodejs.org/en/
### How to run :
- First download the file as zip.
- Extract it in Desktop.
- Open console and type :
1. cd Desktop 
2. cd forikfy-master
3. npm install
4. npm start
